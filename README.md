# myblog
## 基于Vue.js+element-ui实现的个人博客 
学习前端3个月了，看到大佬们都有个人的博客网站了，于是有了这个想法。好的个人博客可以提高自己学习的兴趣，将学到的知识总结到博客上，很有成就感。
### 技术栈主要包括```Vue```,```Vuex```,```axios```,```Vue Router```,```Less```。
## 主要功能
1. 文章归档展示
- 文章标题
- 截取部分文章内容
- 发表事件
- 阅读、点赞人数
- 标签分类
2. 文章分类
3. 书单展示
- 图书封面
- 作者
- 评分
- 心得
3. 文章详情
- 文章内容
- 点赞
- 评论
4. 登录/注册
5. 修改头像
6. 管理员后台管理
- 查看文章列表
- 查看图书列表
- 文章、书单增删改查
- 删除评论
